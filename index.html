This is the home page for questions/feedback about my Android app "Smart Alarm Clock".

Features

  • The alarm clock is configured once and is triggered by default from Monday to Friday. In the settings, the period can be extended to Monday to Saturday or Monday to Sunday.
  • Google Calendar Integration:
    • When activated in the settings (default), the calendar of the logged-in user is used to identify vacation days. The system searches for a configurable keyword in the title - "vacation" by default. Before configuration, logging in and synchronizing the calendar using the provided button are necessary. The two calendars for the next year are automatically updated on December 31st. If vacation days change, they can be manually synchronized again at any time.
    • The public Google calendar is used for the holidays in the country currently set on the smartphone. For Germany, the system also filters for holidays for the selected state in the settings; otherwise, the selection menu is deactivated.
  • The system default ringtone is used. A selection is possible in the settings.
  • An alarm can be turned off or set to "Snooze" for 10 or 30 minutes via a notification on the lock screen. Alternatively, the alarm can also be turned off via the stop button in the app, which opens when the notification is tapped.
  • The next alarm can be skipped once via a button.

Notes

  • The set alarm time and all other settings are automatically saved.
  • In the app's operating system settings, notifications (including on the lock screen) must be allowed.
  • For some manufacturers' devices, additional settings need to be made for the app to prevent the operating system from closing the app and thus deleting the alarm from the system. Please follow the steps for the respective manufacturer at https://dontkillmyapp.com/. In most cases, disabling the power-saving settings for the app is sufficient.
  • The public Google Calendar for holidays must be integrated into the user's Google Calendar. This can be checked in the calendar app via the menu icon. The "Holidays" checkbox must be selected there. Only for Germany: The state must be selected in the settings before synchronization.

Limitations

Multiple alarms cannot be set.
